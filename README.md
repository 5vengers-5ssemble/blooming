## 🌟 Introduction
**음악할동을 하기 경제적으로 어려운 아티스트들을 팬, 투자자들과 연결시켜주는 아티스트 펀딩 플랫폼입니다.**<br>
아티스트들이 자신들을 더 잘 알리고, 사람들과의 소통공간을 마련하기 위해 다음과 같이 3가지 기능을 제공합니다.
### ✨ 앨범, 콘서트 활동 펀딩
### ✨ 몸동작을 인식해 이모지 리액션을 전송할 수 있는 스트리밍 서비스
### ✨ 아티스트 고유의 NFT 등록 및 거래
